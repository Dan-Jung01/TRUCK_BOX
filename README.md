# 트럭 적재 가능 박스 수 계산기 (React + Vite + MUI)

이 프로젝트는 사용자가 입력한 박스 크기를 기준으로 트럭 크기별 적재 가능 박스 수를 계산해주는 간단한 리액트 웹앱입니다.

## 빠른 시작

1) 의존성 설치
```bash
npm install
```

2) 개발 서버 실행
```bash
npm run dev
```

3) 프로덕션 빌드
```bash
npm run build
npm run preview
```

## 기술 스택

- React 18
- Vite
- MUI (@mui/material) + Emotion
